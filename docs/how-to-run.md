# Running DataSentry

DataSentry is a single Windows program — there is nothing to install and no .NET to set up.

1. Download **`DataSentry.exe`** and double-click it. (It sits next to a `tessdata` folder used to read scanned documents; keep the two together.)
2. Paste or browse to the folder you want to check, then click **Scan**.
3. Read the list: each file is marked **Review** (holds personal data — decide for yourself), **Delete** (junk, stale, or duplicate), or **Keep**. Click any row to see why.
4. To clear out the junk, open **Suggested for deletion** and send it to the Recycle Bin — untick anything you want to spare first. Nothing is ever deleted without your say-so, and deletion always goes to the Recycle Bin, so you can get it back.

DataSentry only reports the *type* and *count* of any personal data it finds — "3 emails, 1 IBAN" — never the values themselves.
